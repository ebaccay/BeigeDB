How We Did It

All scraping, parsing, and analysis was completed using Python 3 and/or a Jupyter Notebook. Many packages were also used such as Selenium for scraping, BeautifulSoup4 for parsing, and the Berkeley datascience library with Pandas to generate tables. If you would like to view the source code to this public project, all of the material can be found on the BeigeDB GitHub.

For an even more in-depth analysis on the code as well as any conceptual/underlying ideas regarding anything that we have done, the three BeigeDB data scientists will be creating a paper describing each step in excruciating detail as well possible areas of further analysis. Stay tuned!

In the meantime, if one would like an even more nuanced and fundamental approach as preparation to construct novel analysis, the team wholeheartedly recommends UC Berkeley's introductory/intermediate Computer Science and Data Science courses such as CS 61A, CS 61B, Data 8, and Data 100.
